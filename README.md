# Felicity Event Management System

MERN-based event platform for participants, organizers, and admin.

## Stack
- Frontend: React + Vite + TypeScript + React-Bootstrap
- Backend: Node.js + Express + TypeScript
- Database: MongoDB

## Roles
- participant
- organizer
- admin

## Implemented Scope
- Part 1 core system flows (auth, organizer/admin management, events, participation, tickets, profile/preferences, organizer ops + csv + analytics)
- Selected advanced features are implemented in milestone branches as applicable

## Local Run
1. `cd backend && npm install && npm run dev`
2. `cd frontend && npm install && npm run dev`

## Quality Checks
- Backend: `npm run typecheck`
- Frontend: `npm run build`
